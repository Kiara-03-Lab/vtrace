FAQ

Common questions

Does vtrace work with [my AI tool]?

Yes. vtrace works with ChatGPT, GitHub Copilot, Cursor, Claude, Gemini, and any other AI coding assistant. It captures the conversation and code changes regardless of which tool you use.

Is my code sent to your servers?

No. By default, everything stays on your computer. The Team plan offers optional cloud sync, but you control what gets synced.

How is this different from Git?

Git tracks code changes. vtrace tracks AI conversations AND code changes together ‚Äî so you can see which prompt led to which code. They work great together.

How long does setup take?

About 2 minutes. One install command, and you're ready to go. No configuration needed.
